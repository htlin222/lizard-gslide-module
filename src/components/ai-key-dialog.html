🔑 Set up AI

This feature needs your own Groq API key (free).

  1. Open console.groq.com/keys (free, no card).
  2. Click Create API Key and copy it (starts with gsk_).
  3. Paste it below and press Save & Continue.
🔒 Stored privately on your Google account (UserProperties) — never shared with other users and never sent to the add-on author.
